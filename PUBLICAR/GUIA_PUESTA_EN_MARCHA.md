## PEÑA MAULAS - Web de Gestión de Quinielas

Hola. Ya has configurado Firebase y yo he actualizado todo el código de tu web para que funcione conectado a internet.

Ahora **SOLO FALTA UN PASO**: Subir estos cambios a Internet para que la web se actualice.

### PASO FINAL: Subir los cambios con GitHub Desktop

Sigue estos pasos para que la nueva versión de la web (con Base de Datos) esté disponible para todos:

1.  **Abre GitHub Desktop** en tu ordenador.
2.  Verás que a la izquierda aparece una lista con muchos archivos modificados (`app.js`, `index.html`, `firebase-init.js`, etc.). Esto es normal, son todos los cambios que he hecho yo.
3.  Abajo a la izquierda, donde pone **"Summary (required)"**, escribe algo como: `Conexión con Base de Datos`.
4.  Pulsa el botón azul **"Commit to main"**.
5.  Una vez termine (desaparecerán los archivos de la lista), aparecerá un botón arriba a la derecha que dice **"Push origin"**. Púlsalo.
6.  Espera a que termine de cargar.

¡Y LISTO!
En unos minutos (a veces 1 o 2), la web que ya tenías publicada (`https://tu-usuario.github.io/maulas-web/`) se actualizará sola con la nueva versión.

**¿Cómo sé si funciona?**
1.  Entra en la web desde tu ordenador.
2.  Haz algún cambio (añade un socio de prueba, o cambia un resultado).
3.  Abre la web **desde tu móvil**.
4.  Si ves el cambio que hiciste en el ordenador... **¡FELICIDADES! Ya tienes una Web App real y conectada.**
